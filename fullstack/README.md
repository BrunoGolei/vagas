Objetivo deste teste é avaliar seus conhecimentos em organização, estilo, boas práticas e habilidades em front-end e back-end.

## Requisitos

- HTML5/CSS3/SCSS
- Gulp ou Webpack

- Git
- NodeJS
- APIs / REST
- Cloud (AWS ou Google Cloud)
- Ecmascript 6+
- CDN
- Linux
- Banco de Dados
- PHP
- Suporte para IE9+, Chrome, Safari, Firefox+ :)
